# PDFspliter for Android

![PDFspliter Icon](app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

An Android application to easily split PDF files into multiple smaller documents. This app provides
a simple and intuitive interface for users to select a PDF, choose how to split it, and save the
resulting files directly on their device.

## Features

* **Select PDF files:** Easily browse and select PDF files from your device's storage.
* **Split by page ranges:** Specify custom page ranges to extract from the PDF.
* **Split into single pages:** Quickly split a PDF into individual pages.
* **Save and share:** Save the split PDF files to your device and share them with other apps.
* **Privacy-focused:** The app processes files locally on your device, ensuring your data remains
  private.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Android Studio
* An Android device or emulator

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/marcosnunes/PDFspliter_androidApp.git
   ```
2. Open the project in Android Studio.
3. Build and run the app.
