Instruções

1. Clique em "Selecionar PDF" para escolher um arquivo do seu computador.
2. O nome do arquivo selecionado aparecerá abaixo do botão.
3. Clique no botão "Processar PDF".
4. Aguarde enquanto o sistema processa o seu arquivo e divide em 1 arquivo por cada página do arquivo original.
5. As páginas processadas aparecerão na seção "Páginas Processadas" abaixo, com links para download.
